# The Metaformic Agent: Lived Motion Storyteller 🧠⏳

This project was built for the **Google AI Hackathon (Creative Storyteller Category)**. It is a multimodal agent that uses the Google GenAI SDK and Gemini 2.5 Flash to engineer profound, embodied cognitive resonance by interleaving abstract physics (time dilation) with human biomechanics.

## 🚀 Architecture & Tech Stack
* **Frontend:** Vanilla HTML/JS/CSS ("Cognitive Operating System" UI with asynchronous typewriter pacing).
* **Backend:** Python / Flask (Hosted on Google Cloud Run).
* **AI Model:** Gemini 2.5 Flash via Google GenAI SDK.
* **Grounding:** Custom system instructions based on the "Metaformic Architecture of Lived Motion" and the "Creative Problem Solver" Cardinal Profile.
